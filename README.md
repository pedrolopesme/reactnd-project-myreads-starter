### Running tests

Tests were write using [Jest](https://jestjs.io/) combined with [Enzyme](https://github.com/airbnb/enzyme). In order to run them, just type:

```shell
$ npm test
```

To get current coverage status, just type: 

```shell
$ npm test -- --coverage
```
